About Me

I'm a computer engineering graduate and Python developer, passionate about building real-world web applications.

I’m Micheal Isaac, a Full-Stack Python Developer and graduate of Federal University of Technology, Minna, as well as a Moat Academy alumnus. With over 3 years of experience as an IT System Administrator, I have built and managed websites for companies and collaborated with developers to create both landing pages and full websites.

I specialize in building secure, scalable web applications using Python, Flask, and JavaScript. On the frontend, I create responsive, interactive user interfaces using HTML, CSS, bootstrap, Vanila and React.js to deliver smooth user experiences.

I am experienced with database management, version control using Git/GitHub, and deploying applications to production environments. My focus is on writing clean, maintainable code and developing systems that are easy to understand and extend.
